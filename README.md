### 🚀 What I do

I build production AI systems end to end — from PostgreSQL schemas and FastAPI services, through
RAG pipelines that ground LLMs in thousands of pages of regulation, to React frontends that stream
answers token by token. The result: work that used to take compliance teams hours now takes minutes.

- 🧠 **RAG done right** — retrieval, citations, and guardrails so the model quotes the regulation, not its imagination
- ⚡ **Real-time everything** — WebSockets & SSE streaming, because spinners are where user trust goes to die
- 🔐 **Security first** — Auth0, JWE tokens, PII encryption; compliance software should probably be compliant
- 🧪 **CI as a lifestyle** — GitHub Actions + SonarCloud gatekeeping every merge
